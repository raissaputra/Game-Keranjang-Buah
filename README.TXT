USER INSTRUCTIONS:

- Game keranjang buah ini ada 15 aktor dan 4 latar belakang.
- Aktor keranjang untuk menangkap buah dengan digerakkan ke kanan ke kiri menggunakan tombol anak panah
- aktor akan muncul secara random dari atas ke bawah
- ketika keranjang berhasil menangkap objek buah(apel,anggur,pisang) maka skor tambah 100
- dan jika menangkap objek (pisau1,pisau2) skor berkurang 100 dan nyawa life berkurang 1
- game ini ada 5 nyawa dengan gambar lifebar
- ketika nyawa kurang dari 5,maka nyawa baru(gbr bintang)akan muncul dengan acak
- jika berhasil menangkap  nyawa baru maka nyawa bertambah
- game berakhir jika nyawa habis dan skor kurang dari nol
- user dapat winner jika skornya >2000 dan nyawa blum habis dan berhasil menangkap objek bintang.
